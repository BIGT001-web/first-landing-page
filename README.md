# T-MAJOR'S FOUNDATION — Landing page

Static landing page from the Figma design. Open `landing-page/index.html` in a browser, or serve the folder locally.

```bash
npx serve landing-page
```

## Contents

- `landing-page/index.html` — markup
- `landing-page/styles.css` — styles
- `landing-page/assets/` — images and logo SVG
